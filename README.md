## Key Map

1. b - 바운딩 박스 on / off (타이틀 화면 & 플레이 화면)
2. 방향키 위, 아래 | Enter - 타이틀 화면, 모드 선택 화면
3. 방향키 - 플레이 화면 (이동, 점프, 하강)
4. ESC - 모든 화면 (뒤로 가기)
5. 마우스 좌 - 제작 화면 (드래그 앤 드롭을 통한 맵 제작)
6. 마우스 우 - 제작 화면 (타일 제거)
7. s - 맵 저장

## Play

제작 모드 - 맵 제작 - 로드 모드 - 맵 불러 오기 - 게임 플레이 - 플레이어가 깃발에 무사히 도착 하면 종료!

1. 제작 모드를 통한 맵 제작 (maker mode)
